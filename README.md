# PythonicUtils

Pythonic useful functions and classes for Dart.

## Features

### itertools
Effective iteration.

* next()
* zip()
* enumerate()


### Pair

`Pair` is a class that provides a way to store two objects as a single unit.


### Range

Numeric Sequence


### strtools

Useful functions for String type

* strIter()
* removeSuffix()
* strJoin()
* toString()
